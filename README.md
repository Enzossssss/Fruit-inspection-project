# Fruit-inspection-project

## Requirements
Python and jupyter.

The following packages are needed:
 - ``openCV 4.5.5`` or higher
 - ``matplotlib``
 - ``scipy``
 - ``numpy``
 - ``shapely``

## Description

The overall tasks are developed in ``fruit-inspection.ipynb``.

The ``utils.py`` file contain all used functions.
